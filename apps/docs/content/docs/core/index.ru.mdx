# @ambrosia/core

**@ambrosia/core** — это универсальный высокопроизводительный DI-контейнер (Dependency Injection) для TypeScript и Bun runtime. Обеспечивает мощное управление зависимостями с минимальными накладными расходами и максимальной гибкостью.

- **Singleton (кэшированный):** 1,005,257 операций/сек (0.995μs)
- **Простой сервис (production):** 867,814 операций/сек (1.152μs)
- **Сложный сервис (production):** 443,715 операций/сек (2.254μs)

Production режим на 7.4% быстрее для простых сервисов и на 22.8% быстрее для сложных сервисов по сравнению с development режимом.

## Быстрый пример

```typescript
import { Injectable, Container } from '@ambrosia/core';

@Injectable()
class DatabaseService {
  connect() {
    console.log('Подключено к базе данных');
  }
}

@Injectable()
class UserService {
  constructor(private db: DatabaseService) {}
  
  getUser(id: string) {
    this.db.connect();
    return { id, name: 'Иван Иванов' };
  }
}

// Создаём контейнер
const container = new Container({ mode: 'production' });

// Автоматически разрешаем зависимости
const userService = container.resolve(UserService);
userService.getUser('123');
```

## Почему @ambrosia/core?

### Универсальный DI

В отличие от многих DI-контейнеров, которые тесно связаны с HTTP-фреймворками, **@ambrosia/core** — это чистый DI-контейнер. Используйте его для:

- **CLI-приложений** - Инструменты командной строки с чистым управлением зависимостями
- **Desktop-приложений** - Electron, Tauri приложения с правильной архитектурой
- **Микросервисов** - HTTP, gRPC, потребители очередей сообщений
- **Worker потоков** - Фоновые задачи с dependency injection
- **Тестирования** - Простое mock-объектов и подмена зависимостей

### Оптимизирован для Bun

Создан специально для Bun runtime с:

- **AsyncLogger** - Неблокирующее логирование с `Bun.write()` и `queueMicrotask()`
- **Быстрый запуск** - Оптимизированный поиск провайдеров и кэширование метаданных
- **Нулевые накладные расходы** - Production режим с минимальными runtime проверками
- **Эффективная память** - Умные подсказки GC и управление кэшем

### Production Ready

- **51/51 тестов проходят** - Полное покрытие тестами
- **Набор бенчмарков** - 127+ тестов производительности в 6 наборах
- **Type safe** - Полная поддержка TypeScript в strict режиме
- **Battle tested** - Используется в production микросервисах
