# Установка

Начните работу с @ambrosia/core меньше чем за минуту.

## Требования

<Callout type="info">
  **@ambrosia/core** оптимизирован для **Bun runtime**. Хотя он может работать с Node.js, некоторые функции, такие как AsyncLogger, используют специфичные для Bun API.
</Callout>

**Требования:**
- Bun 1.0.0 или выше
- TypeScript 5.0 или выше (опционально, но рекомендуется)

## Установка Bun

Если у вас ещё не установлен Bun:

<Tabs items={['macOS/Linux', 'Windows']}>
  <Tab value="macOS/Linux">
    ```bash
    curl -fsSL https://bun.sh/install | bash
    ```
  </Tab>
  <Tab value="Windows">
    ```powershell
    powershell -c "irm bun.sh/install.ps1 | iex"
    ```
  </Tab>
</Tabs>

Проверьте установку:

```bash
bun --version
```

## Настройка TypeScript

Настройте TypeScript для поддержки декораторов:

```json title="tsconfig.json"
{
  "compilerOptions": {
    "target": "ES2022",
    "module": "ESNext",
    "moduleResolution": "bundler",
    "experimentalDecorators": true,
    "emitDecoratorMetadata": true,
    "strict": true,
    "skipLibCheck": true,
    "types": ["bun-types"]
  }
}
```

<Callout type="warn">
  **Важно**: Оба параметра `experimentalDecorators` и `emitDecoratorMetadata` должны быть включены для работы декораторов.
</Callout>

## Проверка установки

Создайте тестовый файл для проверки работы:

```typescript title="test.ts"
import { Injectable, Container } from '@ambrosia/core';

@Injectable()
class HelloService {
  greet(name: string) {
    return `Привет, ${name}!`;
  }
}

const container = new Container();
const service = container.resolve(HelloService);

console.log(service.greet('Мир'));
```

Запустите:

```bash
bun run test.ts
```

Ожидаемый вывод:
```
Привет, Мир!
```

<Callout type="success">
  **Успех!** Вы готовы использовать @ambrosia/core.
</Callout>

## Устранение проблем

### Декораторы не работают

**Ошибка:** `experimentalDecorators must be enabled`

**Решение:** Убедитесь, что ваш `tsconfig.json` содержит:
```json
{
  "compilerOptions": {
    "experimentalDecorators": true,
    "emitDecoratorMetadata": true
  }
}
```

### Не удаётся найти модуль

**Ошибка:** `Cannot find module '@ambrosia/core'`

**Решение:** 
1. Проверьте установку: `bun pm ls | grep ambrosia`
2. Переустановите: `bun add @ambrosia/core`
3. Очистите кэш: `rm -rf node_modules && bun install`

### Ошибки времени выполнения с функциями Bun

**Ошибка:** `Bun.write is not a function`

**Решение:** Убедитесь, что вы запускаете с Bun, а не с Node.js:
```bash
bun run index.ts  # ✓ Правильно
node index.ts     # ✗ Неправильно
```
