# Testing WebSocket

`@ambrosia-unce/websocket` provides utilities for testing gateways without starting a real server.

## TestingWsFactory

Creates a WebSocket application with a mock provider:

```typescript title="chat.gateway.test.ts"
import { describe, test, expect } from "bun:test";
import "reflect-metadata";
import { TestingWsFactory, MockWsClient } from "@ambrosia-unce/websocket";
import { ChatGateway } from "./chat.gateway";

describe("ChatGateway", () => {
  test("handles message", async () => {
    const module = await TestingWsFactory.create({
      gateways: [ChatGateway],
    }).compile();

    const client = new MockWsClient("user-1");
    await module.connect("/chat", client);

    await module.send("/chat", client, {
      event: "sendMessage",
      data: { text: "Hello!" },
    });

    // Check that the client received a response
    expect(client.sent).toHaveLength(1);
    expect(client.sent[0].event).toBe("newMessage");
    expect(client.sent[0].data.text).toBe("Hello!");
  });
});
```

## MockWsClient

A mock WebSocket client. Records all sent messages:

```typescript
const client = new MockWsClient("test-user");

// After interacting with the gateway:
client.sent;      // [{ event: "newMessage", data: { text: "Hello" } }]
client.rawSent;   // ["raw data"]
client.closed;    // false
client.data;      // { custom: "data" }
```

## TestingWsModule Methods

```typescript
const module = await TestingWsFactory.create({ gateways: [...] }).compile();

// Simulate a client connection
await module.connect("/chat", client);

// Send a message
await module.send("/chat", client, { event: "ping", data: {} });

// Disconnect
await module.disconnect("/chat", client, 1000, "bye");

// Access DI
const service = module.get(ChatService);

// List registered gateway paths
module.getGatewayPaths(); // ["/chat", "/notifications"]
```

## Overriding Dependencies

```typescript
const mockChatService = {
  addMessage: (text: string) => ({ text, ts: 0 }),
  getHistory: () => [],
};

const module = await TestingWsFactory.create({
  gateways: [ChatGateway],
})
  .overrideValue(ChatService, mockChatService)
  .compile();
```

## Testing Lifecycle

```typescript
test("connection and disconnection", async () => {
  const module = await TestingWsFactory.create({
    gateways: [ChatGateway],
  }).compile();

  const client = new MockWsClient("user-1");

  // Connection
  await module.connect("/chat", client);
  const gateway = module.get(ChatGateway);
  expect(gateway.getConnectedClients()).toContain("user-1");

  // Disconnection
  await module.disconnect("/chat", client);
  expect(gateway.getConnectedClients()).not.toContain("user-1");
});
```

## Best Practices

- Create a new `MockWsClient` for each test
- Check `client.sent` to validate responses
- Use `overrideValue` to mock services
- Test edge cases: sending an unknown event, disconnect without connect
