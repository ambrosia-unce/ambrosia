# Interceptors (Перехватчики)

Interceptors позволяют перехватывать выполнение запроса до и после вызова handler'а. Они идеально подходят для логирования, преобразования данных, кэширования и управления ответами.

## Основы

Interceptor - это класс с методом `intercept()`, который получает `ExecutionContext` и `CallHandler`:

```typescript
import { Injectable } from '@ambrosia-unce/core';
import { Interceptor, ExecutionContext, CallHandler } from '@ambrosia-unce/http';

@Injectable()
export class LoggingInterceptor implements Interceptor {
  async intercept(context: ExecutionContext, next: CallHandler): Promise<any> {
    const { request } = context.switchToHttp();
    const startTime = Date.now();
    
    console.log(`→ ${request.method} ${request.url}`);
    
    // Вызвать следующий interceptor или handler
    const result = await next.handle();
    
    const duration = Date.now() - startTime;
    console.log(`← ${request.method} ${request.url} - ${duration}ms`);
    
    return result;
  }
}
```

## CallHandler

`CallHandler` - это интерфейс для вызова следующего interceptor'а или handler'а:

```typescript
interface CallHandler<T = any> {
  handle(): Promise<T>;
}
```

Вызов `next.handle()` передает управление по цепочке:

```
Request → Interceptor1 → Interceptor2 → Handler → Interceptor2 → Interceptor1 → Response
```

## Применение Interceptors

### На уровне метода

```typescript
import { Controller, Http, UseInterceptor } from '@ambrosia-unce/http';

@Controller('/users')
export class UserController {
  @Http.Get('/')
  @UseInterceptor(LoggingInterceptor)
  async list() {
    return [{ id: 1, name: 'John' }];
  }
}
```

### На уровне контроллера

```typescript
@Controller('/api')
@UseInterceptor(LoggingInterceptor, TransformInterceptor)
export class ApiController {
  // Все методы используют оба interceptor'а
  
  @Http.Get('/data')
  async getData() {
    return { data: [] };
  }
}
```

### Глобальные Interceptors

```typescript
const app = await HttpApplication.create({
  provider: ElysiaProvider,
  globalInterceptors: [LoggingInterceptor, ErrorInterceptor],
  port: 3000
});
```

## Преобразование ответа

Interceptors могут изменять результат handler'а:

```typescript
@Injectable()
export class TransformInterceptor implements Interceptor {
  async intercept(context: ExecutionContext, next: CallHandler): Promise<any> {
    const result = await next.handle();
    
    // Обернуть результат в стандартный формат
    return {
      success: true,
      timestamp: new Date().toISOString(),
      data: result
    };
  }
}

// Ответ:
// {
//   "success": true,
//   "timestamp": "2024-01-15T10:30:00.000Z",
//   "data": { "id": 1, "name": "John" }
// }
```

## Обработка ошибок

Interceptors могут перехватывать и обрабатывать ошибки:

```typescript
@Injectable()
export class ErrorInterceptor implements Interceptor {
  async intercept(context: ExecutionContext, next: CallHandler): Promise<any> {
    try {
      return await next.handle();
    } catch (error) {
      console.error('Ошибка в handler:', error);
      
      // Преобразовать ошибку
      if (error instanceof DatabaseError) {
        throw new InternalServerErrorException('Ошибка базы данных');
      }
      
      // Пробросить дальше для exception filters
      throw error;
    }
  }
}
```

## Dependency Injection

Interceptors поддерживают полную DI:

```typescript
@Injectable()
export class CacheInterceptor implements Interceptor {
  constructor(
    private cacheService: CacheService,
    private configService: ConfigService
  ) {}
  
  async intercept(context: ExecutionContext, next: CallHandler): Promise<any> {
    const { request } = context.switchToHttp();
    const cacheKey = this.generateCacheKey(request);
    
    // Проверить кэш
    const cached = await this.cacheService.get(cacheKey);
    if (cached) {
      console.log(`Cache HIT: ${cacheKey}`);
      return cached;
    }
    
    // Выполнить handler
    const result = await next.handle();
    
    // Сохранить в кэш
    const ttl = this.configService.get('cache.ttl');
    await this.cacheService.set(cacheKey, result, ttl);
    
    return result;
  }
  
  private generateCacheKey(request: IHttpRequest): string {
    return `${request.method}:${request.url}`;
  }
}
```

## Порядок выполнения

Interceptors выполняются в следующем порядке:

```typescript
const app = await HttpApplication.create({
  globalInterceptors: [Global1, Global2], // 1. Глобальные (до)
});

@Controller('/users')
@UseInterceptor(Controller1, Controller2) // 2. Controller (до)
export class UserController {
  @Http.Get('/')
  @UseInterceptor(Method1) // 3. Method (до)
  async list() {
    return []; // 4. Handler
  }
  // 5. Method (после)
  // 6. Controller (после)
  // 7. Global (после)
}
```

Полный цикл:
```
Request
  → Global1 (before)
    → Global2 (before)
      → Controller1 (before)
        → Controller2 (before)
          → Method1 (before)
            → Handler
          ← Method1 (after)
        ← Controller2 (after)
      ← Controller1 (after)
    ← Global2 (after)
  ← Global1 (after)
Response
```

## Паттерны использования

### Timeout Interceptor

```typescript
@Injectable()
export class TimeoutInterceptor implements Interceptor {
  constructor(private readonly timeout: number = 5000) {}
  
  async intercept(context: ExecutionContext, next: CallHandler): Promise<any> {
    const timeoutPromise = new Promise((_, reject) => {
      setTimeout(() => reject(new Error('Request timeout')), this.timeout);
    });
    
    try {
      return await Promise.race([
        next.handle(),
        timeoutPromise
      ]);
    } catch (error) {
      if (error.message === 'Request timeout') {
        throw new InternalServerErrorException('Превышено время ожидания');
      }
      throw error;
    }
  }
}
```

### Performance Monitoring

```typescript
@Injectable()
export class PerformanceInterceptor implements Interceptor {
  constructor(private metricsService: MetricsService) {}
  
  async intercept(context: ExecutionContext, next: CallHandler): Promise<any> {
    const { request } = context.switchToHttp();
    const startTime = Date.now();
    const route = `${request.method} ${context.getClass().name}.${context.getHandler()}`;
    
    try {
      const result = await next.handle();
      const duration = Date.now() - startTime;
      
      // Отправить метрики
      this.metricsService.recordSuccess(route, duration);
      
      return result;
    } catch (error) {
      const duration = Date.now() - startTime;
      this.metricsService.recordError(route, duration);
      throw error;
    }
  }
}
```

### Serialization Interceptor

```typescript
@Injectable()
export class SerializationInterceptor implements Interceptor {
  async intercept(context: ExecutionContext, next: CallHandler): Promise<any> {
    const result = await next.handle();
    
    // Удалить чувствительные данные
    if (result && typeof result === 'object') {
      return this.sanitize(result);
    }
    
    return result;
  }
  
  private sanitize(obj: any): any {
    if (Array.isArray(obj)) {
      return obj.map(item => this.sanitize(item));
    }
    
    if (obj && typeof obj === 'object') {
      const sanitized = { ...obj };
      
      // Удалить чувствительные поля
      delete sanitized.password;
      delete sanitized.passwordHash;
      delete sanitized.secret;
      delete sanitized.token;
      
      // Рекурсивно обработать вложенные объекты
      Object.keys(sanitized).forEach(key => {
        if (sanitized[key] && typeof sanitized[key] === 'object') {
          sanitized[key] = this.sanitize(sanitized[key]);
        }
      });
      
      return sanitized;
    }
    
    return obj;
  }
}
```

### Compression Interceptor

```typescript
@Injectable()
export class CompressionInterceptor implements Interceptor {
  async intercept(context: ExecutionContext, next: CallHandler): Promise<any> {
    const { request, response } = context.switchToHttp();
    const result = await next.handle();
    
    // Проверить Accept-Encoding header
    const acceptEncoding = request.headers['accept-encoding'] || '';
    const shouldCompress = acceptEncoding.includes('gzip') && 
                          this.isCompressible(result);
    
    if (shouldCompress) {
      response.setHeader('Content-Encoding', 'gzip');
      // Логика сжатия будет обработана на уровне HTTP provider
    }
    
    return result;
  }
  
  private isCompressible(result: any): boolean {
    // Проверить размер и тип данных
    const str = JSON.stringify(result);
    return str.length > 1024; // Сжимать только если больше 1KB
  }
}
```

### Conditional Interceptor

```typescript
@Injectable()
export class ConditionalInterceptor implements Interceptor {
  constructor(private configService: ConfigService) {}
  
  async intercept(context: ExecutionContext, next: CallHandler): Promise<any> {
    const isEnabled = this.configService.get('features.interceptor.enabled');
    
    if (!isEnabled) {
      // Пропустить обработку
      return next.handle();
    }
    
    // Выполнить логику
    console.log('Interceptor активен');
    return next.handle();
  }
}
```

## Изменение Request

Interceptors могут модифицировать request перед handler'ом:

```typescript
@Injectable()
export class RequestEnricherInterceptor implements Interceptor {
  async intercept(context: ExecutionContext, next: CallHandler): Promise<any> {
    const { request } = context.switchToHttp();
    
    // Добавить дополнительные данные в request
    request.timestamp = Date.now();
    request.requestId = crypto.randomUUID();
    
    // Логировать с requestId
    console.log(`[${request.requestId}] ${request.method} ${request.url}`);
    
    return next.handle();
  }
}
```

## Композиция Interceptors

Можно создавать составные interceptors:

```typescript
@Injectable()
export class ApiInterceptor implements Interceptor {
  constructor(
    private logging: LoggingInterceptor,
    private transform: TransformInterceptor,
    private error: ErrorInterceptor
  ) {}
  
  async intercept(context: ExecutionContext, next: CallHandler): Promise<any> {
    // Объединить несколько interceptors
    return this.error.intercept(context, {
      handle: () => this.logging.intercept(context, {
        handle: () => this.transform.intercept(context, next)
      })
    });
  }
}
```

## Best Practices

1. **Вызывайте next.handle()**: Всегда вызывайте `next.handle()` для продолжения цепочки
2. **Обрабатывайте ошибки**: Используйте try-catch для обработки ошибок из handler'ов
3. **Минимизируйте overhead**: Interceptors выполняются для каждого запроса - оптимизируйте производительность
4. **Используйте async/await**: Работайте с асинхронными операциями через async/await
5. **Не изменяйте request напрямую**: Лучше прикреплять данные как новые свойства
6. **Документируйте side effects**: Если interceptor изменяет состояние, документируйте это
7. **Тестируйте изолированно**: Пишите unit-тесты для interceptor'ов
8. **Условное выполнение**: Используйте конфигурацию для включения/отключения interceptor'ов
