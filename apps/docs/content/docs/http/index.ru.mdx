# @ambrosia-unce/http

**@ambrosia-unce/http** — это современный высокопроизводительный HTTP фреймворк для TypeScript с мощной системой Dependency Injection. Построен на основе [@ambrosia-unce/core](/docs/core) и предоставляет абстракцию над популярными HTTP библиотеками (Elysia.js, Hono).

## Архитектура жизненного цикла

```
Incoming Request
    ↓
Middleware (@UseMiddleware)
    ↓
Guards (@UseGuard)
    ↓
Interceptors (before)
    ↓
Pipes (validation/transform)
    ↓
Route Handler
    ↓
Interceptors (after)
    ↓
Exception Filters
    ↓
Response
```

## Быстрый пример

```typescript
import { HttpApplication, Controller, Http, Body, UseGuard, Status } from '@ambrosia-unce/http';
import { ElysiaProvider } from '@ambrosia-unce/http-elysia';
import { Type } from '@sinclair/typebox';

// DTO с TypeBox схемой
class CreateUserDto {
  static schema = Type.Object({
    name: Type.String({ minLength: 1 }),
    email: Type.String({ format: 'email' }),
    age: Type.Number({ minimum: 0 })
  });
  
  name!: string;
  email!: string;
  age!: number;
}

// Контроллер с автоматической регистрацией
@Controller('/users')
@UseGuard(AuthGuard)
export class UserController {
  constructor(private userService: UserService) {}
  
  @Http.Get('/')
  async list() {
    return this.userService.findAll();
  }
  
  @Http.Post('/')
  @Status(201)
  async create(@Body() dto: CreateUserDto) {
    // dto автоматически валидируется через ValidationPipe
    return this.userService.create(dto);
  }
}

// Запуск приложения
const app = await HttpApplication.create({
  provider: ElysiaProvider,
  globalGuards: [AuthGuard],
  globalPipes: [ValidationPipe],
  port: 3000
});

await app.listen();
// 🚀 Ambrosia HTTP Server running on http://localhost:3000
```

## Почему @ambrosia-unce/http?

### Производительность без компромиссов

- **Zero-overhead абстракция** - Прямой passthrough к нативному провайдеру
- **WeakMap кэширование** - Все metadata операции кэшируются
- **Handler pre-compilation** - Lifecycle компоненты компилируются один раз
- **Оптимизация параметров** - Специализированные резолверы с фиксированной длиной массива

### Знакомая архитектура

Если вы знакомы с NestJS, вы будете как дома:

- **Те же декораторы** - @Controller, @Get, @UseGuard, @Body
- **Тот же lifecycle** - Middleware → Guards → Interceptors → Pipes → Handler
- **Те же паттерны** - Guards, Interceptors, Pipes, Exception Filters
- **Но легче** - Без модулей, без сложной конфигурации

### Гибкость провайдеров

Один код - разные HTTP библиотеки:

```typescript
// Elysia.js
const app = await HttpApplication.create({
  provider: ElysiaProvider,
  // ...
});

// Hono (в будущем)
const app = await HttpApplication.create({
  provider: HonoProvider,
  // ...
});
```

### TypeBox валидация из коробки

```typescript
class CreatePostDto {
  static schema = Type.Object({
    title: Type.String({ minLength: 5, maxLength: 100 }),
    content: Type.String({ minLength: 10 }),
    tags: Type.Array(Type.String()),
    published: Type.Boolean()
  });
  
  title!: string;
  content!: string;
  tags!: string[];
  published!: boolean;
}

@Controller('/posts')
export class PostController {
  @Http.Post('/')
  async create(@Body() dto: CreatePostDto) {
    // dto уже валидирован!
    return this.postService.create(dto);
  }
}
```

## Основные компоненты

### Декораторы (26 штук)

**Контроллер:**
- `@Controller(path)` - Маркирует класс как контроллер

**HTTP методы:**
- `Http.Get()`, `Http.Post()`, `Http.Put()`, `Http.Delete()`, `Http.Patch()`, `Http.Options()`, `Http.Head()`

**Lifecycle:**
- `@UseGuard()`, `@UseInterceptor()`, `@UsePipe()`, `@UseMiddleware()`, `@UseFilter()`

**Response:**
- `@Status(code)` - Устанавливает HTTP status

**Параметры (11 типов):**
- `@Body()`, `@Query()`, `@Param()`, `@Headers()`, `@Header()`
- `@Req()`, `@Res()`, `@Session()`, `@Cookie()`, `@Ip()`
- `@UploadedFile()`, `@UploadedFiles()`

### Built-in Exceptions

- `BadRequestException` (400)
- `UnauthorizedException` (401)
- `ForbiddenException` (403)
- `NotFoundException` (404)
- `ConflictException` (409)
- `UnprocessableEntityException` (422)
- `InternalServerErrorException` (500)

### Built-in Pipes

- `ValidationPipe` - Автоматическая валидация с TypeBox
- `ParseIntPipe` - Преобразование string → number
- `ParseBoolPipe` - Преобразование string → boolean
