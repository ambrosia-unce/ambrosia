# Installation

Install **@ambrosia-unce/http** and an HTTP provider to get started.

## Prerequisites

- **Bun** >= 1.0.0 or **Node.js** >= 18.0.0
- **TypeScript** >= 5.0.0
- Experimental decorators enabled

## Installing Packages

<Tabs items={['bun', 'npm', 'pnpm', 'yarn']}>
  <Tab value="bun">
    ```bash
    bun add @ambrosia-unce/http @ambrosia-unce/http-elysia @ambrosia-unce/core
    bun add -d @sinclair/typebox
    ```
  </Tab>
  <Tab value="npm">
    ```bash
    npm install @ambrosia-unce/http @ambrosia-unce/http-elysia @ambrosia-unce/core
    npm install --save-dev @sinclair/typebox
    ```
  </Tab>
  <Tab value="pnpm">
    ```bash
    pnpm add @ambrosia-unce/http @ambrosia-unce/http-elysia @ambrosia-unce/core
    pnpm add -D @sinclair/typebox
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn add @ambrosia-unce/http @ambrosia-unce/http-elysia @ambrosia-unce/core
    yarn add --dev @sinclair/typebox
    ```
  </Tab>
</Tabs>

### Packages

- **@ambrosia-unce/core** - DI container (base dependency)
- **@ambrosia-unce/http** - HTTP framework (core functionality)
- **@ambrosia-unce/http-elysia** - Elysia.js provider
- **@sinclair/typebox** - TypeBox for validation (optional but recommended)

## TypeScript Configuration

Update `tsconfig.json` to support decorators:

```json title="tsconfig.json"
{
  "compilerOptions": {
    "target": "ES2022",
    "module": "ESNext",
    "moduleResolution": "bundler",
    "experimentalDecorators": true,
    "emitDecoratorMetadata": true,
    "strict": true,
    "skipLibCheck": true,
    "esModuleInterop": true
  }
}
```

### Important Options

- **experimentalDecorators** - Enables decorator support
- **emitDecoratorMetadata** - Generates metadata for type reflection
- **target** - ES2022 or higher for modern features
- **moduleResolution** - "bundler" for Bun or "node" for Node.js

## Verifying the Installation

Create a simple file for verification:

```typescript title="test.ts"
import { HttpApplication, Controller, Http } from '@ambrosia-unce/http';
import { ElysiaProvider } from '@ambrosia-unce/http-elysia';

@Controller('/')
class AppController {
  @Http.Get('/')
  hello() {
    return { message: 'Hello, Ambrosia!' };
  }
}

const app = await HttpApplication.create({
  provider: ElysiaProvider,
  port: 3000
});

await app.listen();
```

Run it:

<Tabs items={['bun', 'node']}>
  <Tab value="bun">
    ```bash
    bun run test.ts
    ```
  </Tab>
  <Tab value="node">
    ```bash
    npx tsx test.ts
    ```
  </Tab>
</Tabs>

If you see the message:

```
Ambrosia HTTP Server running on http://localhost:3000
```

Congratulations! Installation was successful.

## Project Structure

Recommended structure for an HTTP application:

```
my-app/
+-- src/
|   +-- controllers/
|   |   +-- user.controller.ts
|   |   +-- post.controller.ts
|   +-- services/
|   |   +-- user.service.ts
|   |   +-- post.service.ts
|   +-- guards/
|   |   +-- auth.guard.ts
|   +-- interceptors/
|   |   +-- logging.interceptor.ts
|   +-- pipes/
|   |   +-- validation.pipe.ts
|   +-- dto/
|   |   +-- create-user.dto.ts
|   |   +-- create-post.dto.ts
|   +-- main.ts
+-- tsconfig.json
+-- package.json
```
