# Создание паков

Pack — это набор исходных файлов (сервисы, guards, контроллеры), которые живут внутри вашего проекта. При публикации через registry они копируются как исходный код в проект потребителя.

## Быстрый старт с генератором

<Steps>

### Генерация шаблона

```bash
ambrosia generate pack my-cache
# или короткий алиас
ambrosia g pack my-cache
```

CLI создаст исходные файлы в директории `{packsDir}/my-cache/` вашего проекта (по умолчанию `src/packs/`):

```
src/packs/my-cache/
├── index.ts               ← barrel export
├── my-cache.pack.ts       ← PackDefinition с forRoot/forRootAsync
├── my-cache.service.ts    ← основной сервис
└── types.ts               ← интерфейс конфигурации + InjectionToken
```

<Callout type="info">
  Создаются **только исходные файлы** — никаких package.json, build.ts или README. Pack живёт как часть вашего проекта.
</Callout>

### Реализация логики

Отредактируйте сгенерированные файлы под ваши нужды.

### Публикация (опционально)

Если хотите поделиться паком через registry — см. [Публикация](/docs/cli/registry/publishing).

</Steps>

## Структура pack

Каждый pack следует стандартной структуре Ambrosia:

### types.ts — Типы и токены

Определите интерфейс конфигурации и `InjectionToken`:

```typescript title="src/packs/my-cache/types.ts"
import { InjectionToken } from '@ambrosia-unce/core';

export interface MyCacheConfig {
  /** Адрес Redis */
  host: string;
  /** Порт */
  port: number;
  /** TTL по умолчанию (в секундах) */
  defaultTtl: number;
}

export const MY_CACHE_CONFIG = new InjectionToken<MyCacheConfig>('MyCacheConfig');
```

<Callout type="info">
  `InjectionToken` обеспечивает type-safe инъекцию конфигурации. Строковый идентификатор должен быть уникальным.
</Callout>

### Сервисы — Бизнес-логика

```typescript title="src/packs/my-cache/my-cache.service.ts"
import { Injectable, Inject } from '@ambrosia-unce/core';
import type { MyCacheConfig } from './types';
import { MY_CACHE_CONFIG } from './types';

@Injectable()
export class MyCacheService {
  private store = new Map<string, { value: unknown; expiresAt: number }>();

  constructor(@Inject(MY_CACHE_CONFIG) private config: MyCacheConfig) {}

  async get<T>(key: string): Promise<T | null> {
    const entry = this.store.get(key);
    if (!entry) return null;
    if (Date.now() > entry.expiresAt) {
      this.store.delete(key);
      return null;
    }
    return entry.value as T;
  }

  async set(key: string, value: unknown, ttl?: number): Promise<void> {
    const expiresAt = Date.now() + (ttl ?? this.config.defaultTtl) * 1000;
    this.store.set(key, { value, expiresAt });
  }

  async delete(key: string): Promise<boolean> {
    return this.store.delete(key);
  }
}
```

### Pack — Определение модуля

Pack — это класс со статическими методами `forRoot` и опционально `forRootAsync`:

```typescript title="src/packs/my-cache/my-cache.pack.ts"
import type { PackDefinition, AsyncPackOptions } from '@ambrosia-unce/core';
import { createAsyncProvider } from '@ambrosia-unce/core';
import { MyCacheService } from './my-cache.service';
import type { MyCacheConfig } from './types';
import { MY_CACHE_CONFIG } from './types';

export class MyCachePack {
  /**
   * Синхронная конфигурация — значения известны заранее.
   */
  static forRoot(config: MyCacheConfig): PackDefinition {
    return {
      meta: { name: 'my-cache', version: '0.1.0' },
      providers: [
        { token: MY_CACHE_CONFIG, useValue: config },
        MyCacheService,
      ],
      exports: [MyCacheService],
    };
  }

  /**
   * Асинхронная конфигурация — значения зависят от других сервисов.
   */
  static forRootAsync(options: AsyncPackOptions<MyCacheConfig>): PackDefinition {
    return {
      meta: { name: 'my-cache-async' },
      providers: [
        createAsyncProvider(MY_CACHE_CONFIG, options),
        MyCacheService,
      ],
      exports: [MyCacheService],
    };
  }
}
```

### index.ts — Barrel export

```typescript title="src/packs/my-cache/index.ts"
export { MyCachePack } from './my-cache.pack';
export { MyCacheService } from './my-cache.service';
export type { MyCacheConfig } from './types';
export { MY_CACHE_CONFIG } from './types';
```

## Паттерны

### Interceptor

```typescript title="src/packs/my-cache/cache.interceptor.ts"
import { Injectable } from '@ambrosia-unce/core';
import type { Interceptor, ExecutionContext, CallHandler } from '@ambrosia-unce/http';
import { MyCacheService } from './my-cache.service';

@Injectable()
export class CacheInterceptor implements Interceptor {
  constructor(private cache: MyCacheService) {}

  async intercept(context: ExecutionContext, next: CallHandler) {
    const req = context.switchToHttp().getRequest();
    const key = `${req.method}:${req.url}`;

    const cached = await this.cache.get(key);
    if (cached) return cached;

    const result = await next.handle();
    await this.cache.set(key, result);
    return result;
  }
}
```

### Lifecycle hooks

Pack может использовать lifecycle hooks для инициализации и очистки:

```typescript title="src/packs/my-cache/my-cache.pack.ts"
import type { PackDefinition } from '@ambrosia-unce/core';
import { MyCacheService } from './my-cache.service';
import type { MyCacheConfig } from './types';
import { MY_CACHE_CONFIG } from './types';

export class MyCachePack {
  static forRoot(config: MyCacheConfig): PackDefinition {
    return {
      meta: { name: 'my-cache' },
      providers: [
        { token: MY_CACHE_CONFIG, useValue: config },
        MyCacheService,
      ],
      exports: [MyCacheService],
      onInit: async (container) => {
        const cache = container.resolve(MyCacheService);
        await cache.connect();
      },
      onDestroy: async (container) => {
        const cache = container.resolve(MyCacheService);
        await cache.disconnect();
      },
    };
  }
}
```

### HTTP Pack с контроллерами

Если pack содержит контроллеры, используйте `HttpPackDefinition`:

```typescript title="src/packs/health/health.pack.ts"
import type { HttpPackDefinition } from '@ambrosia-unce/http';
import { HealthController } from './health.controller';
import { HealthService } from './health.service';

export class HealthPack {
  static forRoot(): HttpPackDefinition {
    return {
      meta: { name: 'health' },
      controllers: [HealthController],
      providers: [HealthService],
      exports: [HealthService],
    };
  }
}
```

## Зависимости между паками

Если ваш pack зависит от другого пака из registry, укажите это в `registryDependencies` при публикации:

```json
{
  "name": "rbac-pack",
  "registryDependencies": ["jwt-auth"],
  "ambrosiaDependencies": ["@ambrosia-unce/core", "@ambrosia-unce/http"]
}
```

При установке `ambrosia add rbac-pack` CLI автоматически установит `jwt-auth` первым.

<Callout type="warn">
  Убедитесь, что зависимый pack действительно существует в registry. Циклические зависимости между паками запрещены — CLI выдаст ошибку.
</Callout>

## Именование

Рекомендации по именованию файлов:

| Файл | Назначение | Пример |
|---|---|---|
| `{name}.pack.ts` | PackDefinition | `jwt-auth.pack.ts` |
| `{name}.service.ts` | Основной сервис | `jwt-auth.service.ts` |
| `{name}.guard.ts` | Guard | `jwt-auth.guard.ts` |
| `{name}.interceptor.ts` | Interceptor | `cache.interceptor.ts` |
| `{name}.controller.ts` | Controller | `health.controller.ts` |
| `types.ts` | Типы и токены | `types.ts` |
| `index.ts` | Barrel export | `index.ts` |

## Тестирование

Используйте `TestingPackFactory` из `@ambrosia-unce/core` для тестирования:

```typescript title="src/packs/my-cache/__tests__/my-cache.test.ts"
import { TestingPackFactory } from '@ambrosia-unce/core';
import { MyCachePack } from '../my-cache.pack';
import { MyCacheService } from '../my-cache.service';

describe('MyCachePack', () => {
  let cacheService: MyCacheService;

  beforeEach(async () => {
    const pack = await TestingPackFactory.create(
      MyCachePack.forRoot({
        host: 'localhost',
        port: 6379,
        defaultTtl: 60,
      })
    );
    cacheService = pack.get(MyCacheService);
  });

  it('should set and get values', async () => {
    await cacheService.set('key', 'value');
    const result = await cacheService.get('key');
    expect(result).toBe('value');
  });

  it('should return null for expired keys', async () => {
    await cacheService.set('key', 'value', 0);
    const result = await cacheService.get('key');
    expect(result).toBeNull();
  });
});
```

Запуск тестов:

```bash
bun test
```
