# @ambrosia/cli

**@ambrosia/cli** — это инструмент командной строки для Ambrosia, который позволяет создавать проекты, генерировать ресурсы и подключать **готовые модули (packs)** из registry прямо в исходный код вашего проекта.

## Философия

В отличие от классических npm-пакетов, **Pack Registry** работает по модели **code distribution** — как [shadcn/ui](https://ui.shadcn.com) для React-компонентов. Когда вы добавляете pack, его исходный код копируется в ваш проект. Вы **владеете кодом** и можете свободно его модифицировать.

```
npm-пакет:     node_modules/@lib/auth  →  закрытый, обновляется через npm
ambrosia pack: src/packs/jwt-auth/     →  ваш код, полный контроль
```

## Быстрый пример

```bash
# Создайте проект
ambrosia new my-app

# Инициализируйте pack registry
cd my-app
ambrosia init

# Добавьте pack
ambrosia add jwt-auth
```

После выполнения `ambrosia add jwt-auth` в вашем проекте появится:

```
my-app/
├── src/
│   ├── packs/
│   │   └── jwt-auth/
│   │       ├── index.ts
│   │       ├── jwt-auth.pack.ts
│   │       ├── jwt-auth.service.ts
│   │       ├── jwt-auth.guard.ts
│   │       └── types.ts
│   └── main.ts
├── ambrosia.json
└── package.json
```

Подключите pack в вашем приложении:

```typescript title="src/main.ts"
import { HttpApplication } from '@ambrosia/http';
import { ElysiaProvider } from '@ambrosia/http-elysia';
import { JwtAuthPack } from './packs/jwt-auth';

const app = await HttpApplication.create({
  provider: new ElysiaProvider(),
  packs: [
    JwtAuthPack.forRoot({
      secret: process.env.JWT_SECRET!,
      expiresIn: '1h',
    }),
  ],
});

await app.listen(3000);
```
